# 🎵 Spotify Clone

A responsive **Spotify Clone** built using **HTML, CSS, and JavaScript**. This project recreates the core user interface and music player functionality of Spotify, including playlists, music playback, seekbar controls, volume adjustment, and a responsive layout.

---

## 🚀 Features

- 🎧 Play and Pause Music
- ⏮️ Previous & Next Track Controls
- 📂 Dynamic Playlist Loading
- 📀 Album Cards
- 🎵 Song List
- ⏱️ Live Song Timer
- 🎚️ Interactive Seekbar
- 🔊 Volume Control
- 🔇 Mute / Unmute
- 📱 Fully Responsive Design
- 🍔 Mobile Hamburger Menu

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/shaikh123-SGH/spotify-clone.git
```

2. Open the project folder.

3. Start a local server (recommended).

For VS Code:

- Install the **Live Server** extension.
- Right-click **index.html**
- Click **Open with Live Server**
